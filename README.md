# Spring Boot 3

## Objetivos 
- Desenvolvimento de uma API Rest
- CRUD (Create, Read, Update e Delete)
- Validações
- Paginação e ordenação
- Implementação de autorização com roles
- Criação de endpoint para redefinição de senha
- Refinamento de validações

## Tecnologias
- Spring Boot 3
- Java 17
- Lombok
- MySQL/ Flyway
- JPA/ Hibernate
- Maven
- Postman
- Swagger
